🏫 学校信息

了解公立高中国际部和民办国际学校的招生、录取和校内活动信息：

🎯 公立高中国际部

📋 招生要求

  • 中考成绩：通常要求达到当地重点高中分数线
  • 英语水平：需要通过英语笔试和面试，雅思/托福成绩优先
  • 综合素质：课外活动、竞赛经历、领导力等
  • 户籍限制：部分学校可能有本地户籍要求
  • 面试表现：展示英语沟通能力和个人特质

📝 录取流程

1. 报名阶段（3-5月）：提交报名表、成绩单、获奖证书等材料

2. 笔试阶段（5-6月）：参加英语、数学、理化等科目考试

3. 面试阶段（6月）：中英文面试，考察综合素质

4. 中考成绩（7月）：结合中考成绩和笔试面试成绩综合评定

5. 录取通知（7-8月）：发放录取通知书，办理入学手续

🎉 校内活动

  • 学术竞赛：AMC数学竞赛、物理碗、化学竞赛等
  • 社团活动：模拟联合国、辩论社、科学社、艺术社等
  • 体育活动：篮球、足球、游泳、网球等校队
  • 志愿服务：社区服务、环保活动、支教等
  • 国际交流：海外游学、交换生项目、国际会议
  • 科研项目：与大学合作的科研项目、创新实验室

🏆 代表性学校

北京：人大附中ICC、北京四中国际部、十一学校国际部
上海：上海中学国际部、复旦附中国际部、交大附中国际部
广州：华附国际部、省实国际部、执信国际部
深圳：深中国际部、深外国际部、实验中学国际部

🌟 民办国际学校

📋 招生要求

  • 入学考试：自主命题的英语、数学、逻辑等科目考试
  • 英语水平：要求较高，通常需要雅思6.0+或托福80+
  • 综合素质：重视课外活动、领导力、创新能力
  • 无户籍限制：面向全国招生，部分学校招收外籍学生
  • 家长面试：部分学校要求家长参加面试

📝 录取流程

1. 开放日参观（全年）：预约参加校园开放日，了解学校情况

2. 在线报名（10月-次年4月）：提交申请表、成绩单、个人陈述等

3. 入学考试（11月-5月）：参加学校组织的入学考试

4. 面试评估（考试后1-2周）：学生面试，部分学校有家长面试

5. 录取通知（滚动录取）：考试后2-4周发放录取结果

6. 缴费注册（录取后2周内）：缴纳学费定金，完成注册

🎉 校内活动

  • 高端竞赛：国际奥赛、ISEF科学展、哈佛辩论赛等
  • 特色社团：机器人社、投资俱乐部、创业社、艺术工坊
  • 体育竞技：高尔夫、击剑、赛艇、马术等高端项目
  • 社会实践：国际志愿者、海外实习、公益项目
  • 国际认证：CAS活动项目、EPQ研究项目
  • 领导力培养：学生会、社团社长、项目组长等

🏆 代表性学校

北京：北京顺义国际学校、北京哈罗、北京德威
上海：上海包玉刚、上海平和、上海星河湾
广州：广州贝赛思、广州NCPA、广州ULC
深圳：深圳国际交流学院、深国交、深圳贝赛思

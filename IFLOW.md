# 千橙平台 - 项目上下文

## 项目概述

**项目名称**: 千橙平台 (qiancheng)  
**项目类型**: 静态网站 / 微信公众号内容平台  
**目标用户**: 关注孩子未来的中学生家长  
**核心价值**: 提供系统的中学生出国留学资讯，帮助家庭规划国际化教育路径

### 项目目标

为需要留学的家庭提供完整的申请流程介绍、注意事项、IB/AP课程介绍、院校选择、费用分析、申请案例、学校信息和机构推荐等服务，帮助家长和孩子做出明智的留学决策。

### 技术栈

- **前端**: 纯 HTML + CSS (无 JavaScript 框架)
- **部署**: GitHub Pages (通过 `.nojekyll` 文件支持)
- **版本控制**: Git

## 项目结构

## 核心功能模块

### 1. 申请流程
涵盖英国、美国、澳大利亚、新西兰、加拿大、香港等国家和地区的申请流程、申请系统、成绩要求和入学时间。

### 2. 院校选择
提供各国顶尖院校数据库，包括英国 G5、美国常春藤、加拿大八大、澳洲八大等。

### 3. 课程介绍
详细介绍 7 种国际高中课程体系：
- IB (国际文凭课程)
- AP (美国大学先修课)
- A-Level (英国高中课程)
- DSE (香港中学文凭)
- IGCSE (国际中等教育)
- VCE (澳大利亚课程)
- OSSD (加拿大课程)

### 4. 申请案例
提供真实的成功申请案例，包括学生背景、成绩、成功要点等。

### 5. 学校信息
对比公立高中国际部和民办国际学校的：
- 招生要求
- 录取流程
- 校内活动
- 代表性学校

### 6. 注意事项
时间规划、语言考试、文书准备、住宿安排、签证申请等关键提醒。

### 7. 费用分析
详细的费用 breakdown：
- 高中国际部/国际学校费用（3年）
- 海外本科四年费用
- 申请辅助费用（可选）
- 总投资参考

### 8. 机构推荐
推荐知名留学咨询机构，包括新东方前途出国、启德教育、金吉列留学等。

## 设计规范

### 颜色方案
- 主色调: `#667eea` (紫色渐变起始色)
- 辅助色: `#764ba2` (紫色渐变结束色)
- 背景色: `#f8f9fa` (浅灰)
- 文本色: `#333` (深灰)

### 布局特点
- 响应式设计，支持移动端
- 最大宽度: 1200px
- 卡片式布局，带阴影和悬停效果
- 网格布局自适应屏幕尺寸

### 导航结构
固定顶部导航栏，包含 9 个快速锚点链接：
- 首页
- 申请流程
- 院校选择
- 课程介绍
- 申请案例
- 学校信息
- 注意事项
- 费用分析
- 机构推荐

## 开发约定

### 文件命名
- 使用小写字母和连字符
- HTML 文件使用语义化命名

### 代码风格
- HTML 使用缩进（4 空格）
- CSS 使用内联样式（当前实现）
- 注释使用中文

### 内容更新
- 所有留学相关数据需保持准确性和时效性
- 费用信息需标注为估算值
- 机构推荐需保持中立客观
